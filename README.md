# AI Course Final Project

## Linear Regression Prediction App

This project is a Streamlit web application that uses a trained Linear Regression model to make predictions.

## Technologies

- Python
- Streamlit
- Scikit-learn
- NumPy
- Joblib

## Project Files

- `app.py` - Streamlit user interface
- `model.py` - model training and preparation
- `linear_regression_model.joblib` - trained regression model
- `X.joblib` - input dataset
- `y.joblib` - target values
- `requirements.txt` - required Python libraries

## How to Run Locally

Install dependencies:
pip install -r requirements.txt

Run application:
streamlit run app.py
